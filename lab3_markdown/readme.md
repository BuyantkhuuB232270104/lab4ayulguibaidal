Inner join: Хоёр хүснэгтэд хоёуланд нь байгаа мэдээллийг гаргана. Left join: Зүүн талын хүснэгтийн бүх мэдээллийг авна. Хэрэв баруун талын хүснэгтэд тохирох өгөгдөл байхгүй бол NULL гэж харуулна. Захиалгагүй үйлчлүүлэгчид: Зөвхөн захиалга өгөөгүй хэрэглэгчдийг шүүж олоход ашиглана. Right join: Баруун талын хүснэгтийн бүх мэдээллийг авна. зүгээр л left join-ийн эсрэг нь гэсэн үг л дээ. Full join: Хоёр хүснэгтийн бүх мэдээллийг нэгтгэж харуулна. Тохирохгүй хэсгүүд нь NULL байна. Self join: Хүснэгтийг өөртэй нь холбоно. Join + where: join-оор нэгтгэж where-ээр шүүж авна. Олон хүснэгт join: Гурав болон түүнээс дээш хүснэгтийг дамжуулан холбоно. Cross join:Хоёр хүснэгтийн бүх боломжит хувилбарыг гаргана. Хэрэглэгч бүр дээр бүтээгдэхүүн бүрийг харуулна гэсэн үг.
